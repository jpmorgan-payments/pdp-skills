J.P. Morgan Payments Skills

Agent Skills that provide a conversational experience to walk you from getting started with J.P. Morgan Payments APIs to integrating with new ones and expanding your integration.

This is a set of Agent Skills that give GitHub Copilot, Claude Code, Cursor, and other Agent harnesses procedural knowledge of J.P. Morgan's Payments Developer Portal (PDP) APIs. Once installed, the agent can guide you through getting started, generate a working OAuth module, integrate Checkout or Online Payments, and help you expand the integration with new APIs over time — without you having to re-read the docs.

What's Included

Skill What It Does
jpm-integrations-get-started Get started with J.P. Morgan Payments. Acknowledges the disclaimer, captures your goal, confirms requirements to get started, and prepares your project's .env.
jpm-oauth Generates a working OAuth (JWT signing + IDAnywhere token exchange) module in your project, with token caching baked in.
jpm-merchant-integrations Walks you through one API integration at a time, then helps you expand by adding new APIs. Initial release supports Checkout and Online Payments.

The skills chain together — each hands off to the next when ready — and each also works on its own.

API Coverage

API Capabilities
Checkout Drop-in UI, Hosted Payments Page, checkout sessions, capture methods
Online Payments Auth, capture, void, refund; card, wallet, and APM payment methods; 3-D Secure
More APIs coming soon — Tokenization, Reporting, and others are planned for future releases.

Supported Agent Harnesses

Any harness that supports the Agent Skills open format. Check your harness's docs for the canonical skills directory.
